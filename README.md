# LUKSOMER LLC - Корпоративный сайт

Полноценный HTML/CSS/JS сайт с мультиязычностью (RU, UZ, EN) для компании LUKSOMER LLC.

## Структура проекта

```
Luksomer_html/
├── index.html              # Главная страница
├── about.html              # О компании
├── projects.html           # Список проектов
├── project-detail.html     # Детальная страница проекта
├── contacts.html           # Контакты
├── css/
│   └── styles.css          # Все стили сайта
├── js/
│   ├── i18n.js            # Мультиязычность
│   ├── main.js            # Основная логика
│   └── projects.js        # Логика для страницы проектов
└── README.md              # Этот файл
```

## Особенности

✅ **Мультиязычность**: 3 языка (Русский, Узбекский, Английский)
✅ **Адаптивный дизайн**: Работает на всех устройствах
✅ **Плавные анимации**: Анимации при скролле
✅ **SEO-оптимизация**: Мета-теги и семантическая верстка
✅ **Быстрая загрузка**: Оптимизированные изображения
✅ **Кросс-браузерность**: Работает во всех современных браузерах

## Функционал

### Мультиязычность
- Переключение языков без перезагрузки страницы
- Сохранение выбранного языка в localStorage
- Автоматический перевод всех элементов

### Интерактивность
- Мобильное меню
- Анимации при скролле
- Валидация форм
- Фильтрация проектов
- Кнопка "Вверх"
- Параллакс эффект
- Счетчики статистики

### Формы
- Валидация email и телефона
- Сообщения об успешной отправке
- Сброс формы после отправки

## Установка

1. Скачайте папку `Luksomer_html`
2. Откройте `index.html` в браузере
3. Готово!

Для загрузки на сервер:
1. Загрузите все файлы на хостинг
2. Убедитесь, что `index.html` находится в корневой директории
3. Проверьте права доступа к файлам

## Кастомизация

### Изменение цветов
Откройте `css/styles.css` и измените CSS переменные в `:root`:

```css
:root {
    --primary-color: #2563eb;
    --dark: #0f172a;
    --gray-light: #f1f5f9;
    /* ... */
}
```

### Добавление переводов
Откройте `js/i18n.js` и добавьте новые ключи в объект `translations`.

### Изменение контента
Отредактируйте HTML файлы. Для мультиязычных элементов используйте атрибут `data-i18n`.
